Hola, aquí pueden encontrar un Beamer de LaTex, el cual pueden modificar como ustedes quieran, al igual que pueden cambiarle el tamaño, colres, etc.

Recuerden que deben descargar el Tema.sty, ya que pues es el estilo del beamer jajaja

Algunas fuentes puede que no las tengan, tienen que descargarlas o cambiarlas.

Si les da error, es por las imagenes que pues no subí, solo borrenlas, comentenlas o suban las de ustedes.

Los logos los encuentran en la carpeta: img

Recomiendo usar colores de esta página, es muy buena: https://latexcolor.com
